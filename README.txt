I) How To Play: (sound_surfer.exe is located in the dist folder)

Note-
If this is your first time playing, please add .wav music 
files into the same folder as the sound_surfer.exe
After songs have been added to the program's database,
you are no longer required to keep them in your folder.

1. Press 1 or 2 for inputting a new song or selecting an already existing song
2. Select a difficulty setting (E - Easy, M - Medium, H - Hard)
3. Wait for the loading process to complete, then press enter
4. Controls are simple, use the mouse to control the play button and dodge the sound waves
5. If you win/loose, input a name and an optional quote to taunt your challengers!

#################################################################################################
#################################################################################################

II) Running the .py file:

Dependencies - pygame, numpy
1. Locate sound_surfer.py, this is the main game file

III) Compiling to exe:
Dependencies - pygame, numpy, and py2exe
1. If you would like to compile the game into an exe, please install the dependencies and run the
pygame2exe.py file with the commandline ('C:\\...\\python pygame2exe.py')
